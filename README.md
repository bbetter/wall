# wall

Simple Wallpaper Engine launcher for Linux.

`wall` lets you use your **Steam Wallpaper Engine workshop wallpapers** on Linux using  
`linux-wallpaperengine`.

It includes:

- random wallpaper daemon
- workshop browser with tags
- fuzzy search via fuzzel
- KDE / Hyprland friendly setup

---

# Features

- launch Wallpaper Engine wallpapers on Linux
- daemon that keeps wallpaper running
- random wallpaper selection
- browse wallpapers with `wall-picker`
- search by **title** or **tags**
- multi-monitor support
- works on **Hyprland**, **KDE**, and other Wayland compositors

---

# Requirements

- Arch Linux
- yay
- Steam with Wallpaper Engine installed
- workshop wallpapers downloaded

Dependencies installed automatically:

- linux-wallpaperengine
- jq
- fuzzel
- mpvpaper
